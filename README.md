## 📦 Instalação e Uso

### 🌐 Acesso Online (Recomendado)
Acesse a plataforma diretamente:

```bash
https://lucsdesignerf.github.io/curso-python-2026
```

### 💻 Execução Local

```bash
# Clone o repositório
git clone https://github.com/LucsDesignerF/curso-python-2026.git

# Acesse a pasta
cd curso-python-2026

# Sirva os arquivos localmente
# Opção 1: Python
python -m http.server 8080

# Opção 2: Node.js
npx serve .

# Opção 3: VS Code Live Server
# Instale a extensão e clique em "Go Live"

# Acesse no navegador
# http://localhost:8080
```

### ☁️ Deploy no Cloudflare Pages

1. Faça o push para o GitHub
2. Conecte o repositório ao Cloudflare Pages
3. Configure:
   - **Build command:** (deixe em branco)
   - **Build output directory:** `/`
4. Deploy automático a cada push
